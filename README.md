# Personal Portfolio Homepage

**Student Name:** Tooba Butt

## Objective

This project was built as the Day 1 assignment for the TechTide Summer Internship 2026. The
objective is to practice writing clean, semantic HTML and organizing a webpage properly,
using HTML only (no CSS or JavaScript).

## Technologies Used

- HTML5 (semantic elements only — no CSS or JavaScript)

## Features Implemented

- Semantic document structure using `<header>`, `<nav>`, `<section>`, `<address>`, and `<footer>`
- Proper heading hierarchy (`<h1>` – `<h3>`)
- Hero section with a profile image and introduction
- About Me section with work experience and projects, using ordered and unordered lists
- Skills section organized into categorized lists
- Education section with degree details
- Contact Information section using the `<address>` element with a clickable email link
- Navigation bar with anchor links to every section on the page
- Footer with copyright information

## Project Structure

```
portfolio-project/
├── index.html
├── images/
│   └── profile.svg
└── README.md
```

## How to Run the Project

1. Clone or download this repository.
2. Open `index.html` directly in any web browser (no build step or server required).
